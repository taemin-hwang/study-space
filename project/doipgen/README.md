# doipgen
UDS/DoIP packet generator

## Dependencies
+ A C++11 enabled compiler like gcc >= 4.8 is needed.
+ doipgen uses CMake as buildsystem.

## Ubuntu 14.04:
```
sudo apt-get install cmake
sudo apt-get install -y rapidjson-dev
```

## Build
For build call:
```
source ./run_build.sh
```

## Clean
For clean call:
```
source ./run_clean.sh
```
